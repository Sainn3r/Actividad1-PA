# Actividad 1 
## Archivos

**ejercicio1.py:** Maneja la información de estudiantes (nombre, edad, estado, materias y notas). Calcula el promedio, muestra la materia con mejor y peor nota y determina si el estudiante aprueba.

**Ejecución:**
python ejercicio1.py

**ejercicio2.py:** Recibe un valor monetario y calcula el impuesto del 19%, aplica un descuento del 10% si corresponde y muestra el total final.

**Ejecución:**
python ejercicio2.py

**ejercicio3.py:** Genera la serie de Fibonacci según el número de iteraciones ingresado. Muestra la lista generada, la cantidad de términos y el último valor.

**Ejecución:**
python ejercicio3.py

**ejercicio4.py:** Presenta un menú donde el usuario puede elegir entre determinar si un número es primo, calcular el factorial de un número o contar las vocales en un texto.

**Ejecución:**
python ejercicio4.py

**ejercicio5.py:** Solicita nombre, edad y correo electrónico, valida que los datos sean correctos y guarda la información en una lista.

**Ejecución:**
python ejercicio5.py
